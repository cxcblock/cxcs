# CXCS



## Linux 节点使用指南

### 安装

``` shell
 下载 https://github.com/cxcblock/cxcs/archive/master.zip
 解压 master.zip并进入
 为文件加执行权限 sudo chmod +x install.sh
 执行：./install.sh
```
### 启动

#### 启动正式链节点

` ./cxcsz CXCChain` 

### CLI交互方式管理节点

#### 管理正式链节点

` ./cxcsi CXCChain `

### 停止节点

#### 停止正式链节点

` ./cxcsi CXCChain stop `
